# `assets/` — ressources statiques

| Emplacement | Contenu |
|---|---|
| racine (`logo.png`, `react.svg`) | Marque principale de l'application |
| `logos/` | Logos d'institutions (IFRI, EPAC, FSS…) |
| `screenshots/` | Captures d'écran affichées sur la landing page (`pages/Landingpage.jsx`) |

Convention : ne pas déposer ici de fichiers de travail/jetables. Tout asset présent doit être importé quelque part.
